#### setup

Require:
1. python >= 3.8
2. java >=8
3. poetry

To install:

- modify setup.sh to include bot token and your id.
- check to see if prerequisites stated above are installed.
- check to see if commands ``poetry``, ``javac``, ``java``, ``python3`` are installed.
- run ``bash setup.sh``
